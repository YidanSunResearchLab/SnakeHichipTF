# ATACfootprinting: ATAC-Seq Footprinting

`ATACfootprinting` analyzes ATAC-seq BAM files for transcription factor footprints using the TOBIAS software on enhancer regions.


## Usage
```bash
ATACfootprinting [options]
```

### Required Arguments
- `-o/--output_dir`: Output directory.
- `--organism`: Genome assembly, the same as Genomesetup step (e.g., `hg38`).
- `--genome_dir`: Directory from Genomesetup step (e.g., `/abs/path/to/genome`).
- `--atacseq_bam`: Space-separated filtered BAM files (e.g., `"alpha27_atac.filtered.bam GM12878_atac.filtered.bam"`).
- `--enhancer_bed`: BEDPE file with interactions, derived from Hichipsnake step (e.g., `"maps_out/GM12878.significantInteractions.bedpe"`).
- `--samples`: Space-separated sample names matching BAMs (e.g., `"Alpha27 GM12878"`).

### Optional Arguments
- `--threads`: CPU threads (e.g., `100`). Default: 30.
- `--motif`: Path to motif file or directory (e.g., JASPAR PFM). Default: JASPAR MEME downloaded Mar, 10th 2025.
- `--atacorrect`, `--footprinting`, `--bindetect`, `--plot_aggregate`, `--plot_changes`: TOBIAS-specific options (see TOBIAS docs).

### Example
```bash
ATACfootprinting \
  -o /path/to/hichipsnake/step/output/directory/ \
  --organism hg38 \
  --atacseq_bam "sample1.bam sample2.bam" \
  --enhancer_bed "sample1.significantInteractions.bedpe" \
  --samples "sample1 sample2" \
  --threads 100 \
  --motif "/path/to/JASPAR2024_CORE_non-redundant_pfms_meme03142025"
```

## ATACfootprinting Output
```
hichip_test_samples_output/
├── Alpha27/
│   ├── ATACorrect/
│   │   └── Alpha27_corrected.bw
│   ├── Footprints/
│   │   └── Alpha27_footprints.bw
│   ├── BINDetect/
│   │   └── motifs/
│   └── Plots/
│       └── aggregate_footprints.pdf
├── GM12878/
│   ├── ATACorrect/
│   ├── Footprints/
│   ├── BINDetect/
│   └── Plots/
└── logs/
    └── atac_footprinting.{sample}.log
```
- **`ATACorrect/`**: Bias-corrected bigWig files.
- **`Footprints/`**: Footprint score bigWig files.
- **`BINDetect/`**: TF binding predictions per motif. This is one of the final result folders generated by the pipeline.
- **`Plots/`**: Aggregate footprint visualizations. This is one of the final result folders generated by the pipeline.

## Notes
- BAMs must be filtered and coordinate-sorted, refer to snakepipes.
- Ensure `--samples` matches the order of `--atacseq_bam`.
